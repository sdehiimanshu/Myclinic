# 🏥 MyClinic - ASP.NET Core Web Application

## 📌 Project Overview
MyClinic is a Clinic Management System built using ASP.NET Core following Onion Architecture.  
It helps manage patient records, appointments, and contact queries efficiently.

---

## 🏗️ Architecture
This project is built using **Onion Architecture** to ensure clean separation of concerns:

- **Domain Layer** → Entities & Core Business Logic  
- **Application Layer** → DTOs, Interfaces, Services  
- **Infrastructure Layer** → Database, Repositories  
- **Web Layer** → UI (MVC), Controllers  

---

## 🚀 Features
- Patient Management  
- Appointment Booking System  
- Contact Query Handling  
- Clean Architecture Implementation  
- Dependency Injection
